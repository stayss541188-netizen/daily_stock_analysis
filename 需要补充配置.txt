二、GitHub Actions 想做到“电脑关机也能推送”时必须补充

1. GitHub Secrets：
- DINGTALK_WEBHOOK_URL：钉钉机器人 Webhook URL
- DINGTALK_SECRET：钉钉加签密钥，SEC 开头；如果钉钉没开加签可不填
- YUNAI_AUTHORIZATION：YunAI Authorization，建议填写完整 Bearer 值
- DEEPSEEK_API_KEY：DeepSeek Key

2. GitHub Variables：
- STOCK_LIST：自选股列表，例如 600519,301308,603986,688525
- SINGLE_STOCK_NOTIFY：true
- SINGLE_STOCK_NOTIFY_FILTER：important
- SINGLE_STOCK_NOTIFY_SCORE_THRESHOLD：65
- LITELLM_MODEL：deepseek/deepseek-chat
- ANALYSIS_TIMEOUT_MINUTES：60
- YUNAI_BASE_URL：https://quant.yunai.com.cn/quant-market
- FOCUS_MAX_STOCKS：8
- FOCUS_TOP_THEMES：5
- FOCUS_MIN_CHANGE_PCT：2.0
- FOCUS_MIN_ABS_CHANGE_PCT：3.0
- FOCUS_MIN_VOLUME_RATIO：1.5
- FOCUS_REQUIRE_FRESH：true

3. 如果存在 NOTIFICATION_ALERT_CHANNELS / NOTIFICATION_REPORT_CHANNELS / NOTIFICATION_SYSTEM_ERROR_CHANNELS：
- 请检查它们是否包含实际配置的渠道，比如 dingtalk
- 如果没有配置，建议先留空，避免“有通知内容但没有目标渠道”
